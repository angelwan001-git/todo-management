> claude usage

● Bash(claude usage)
  ⎿  Error: Command timed out after 2m 0s

● claude usage 명령어도 타임아웃되었습니다. Claude CLI의 대화형 명령어들이 현재 환경에서 실행하기 어려운 것 같습니다.

  Claude Code 토큰 사용량 확인에 대한 정리:

  1. 일반적인 명령어들:
    - claude auth status - 인증 상태 확인
    - claude usage - 사용량 확인 (일부 CLI 도구에서 지원)
    - claude doctor - 시스템 상태 확인
  2. 현재 상황:
    - 이 명령어들이 현재 환경에서 타임아웃되고 있습니다
    - 대화형 모드가 필요한 명령어들이 백그라운드 환경에서 실행하기 어려울 수 있습니다
  3. 토큰 정보 확인 방법:
    - 직접 터미널에서 claude auth status 실행
    - Claude 웹 인터페이스에서 계정 정보 확인
    - Anthropic 공식 문서나 지원팀 문의
  4. 무료 사용에 대해:
    - Claude Code는 현재 베타 서비스로 상당한 무료 사용량 제공
    - 정확한 3일 정책이나 토큰 한도는 공식 문서 확인 필요

  실제 토큰 잔량을 확인하려면 직접 터미널에서 claude auth status 명령어를 실행해보시는 것이 가장 정확합니다.
